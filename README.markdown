# Rails Photo - Projekt serwisu z galeriami zdjęć.

## Już działa:

* Dodany szablon css
* Dodawanie i oglądanie zdjęć (trzeba zainstalować imagemagick oraz gem paperclip)
* Logowanie
## Do zrobienia:

* Tworzenie galerii
* Dodawanie zdjęć do galerii
* Komentarze do zdjęć
* Grupy użytkowników
* Tagowanie zdjęć
* Kategorie zdjęć
* System wiadomości
* Geolokalizacja
